# 🚀 Soft Jobs API

Backend desarrollado con Node.js, Express y PostgreSQL para la autenticación y autorización de usuarios utilizando JWT.

Este proyecto forma parte del desafío **Soft Jobs**.

---

## 🛠 Tecnologías utilizadas

- Node.js
- Express
- PostgreSQL
- JWT (jsonwebtoken)
- bcrypt
- dotenv
- cors
